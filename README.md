# Docker-for-IPS

To use these:
1. Clone this repository.
2. Run the following command in the terminal:
docker-compose up --d --build
3. Once up, you can register, log in, and check your semester grade point average.
4. If you have disabled or closed Visual Studio Code and Docker Desktop, Docker-compose will be down.
5. If it is down, run the following command in the terminal to update the status:
docker-compose up -d
